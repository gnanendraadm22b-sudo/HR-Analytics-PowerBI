# 📊 HR Analytics Dashboard | Employee Engagement Analysis

---

# 📌 Project Overview

This project presents an **interactive HR Analytics Dashboard** built using **Power BI**, **DAX**, and **Power Query** to analyze employee engagement survey responses across multiple departments and organizational roles.

The dashboard enables HR leaders and management teams to monitor employee engagement, identify organizational strengths and improvement areas, compare departmental performance, and evaluate survey participation using interactive visualizations and KPI-driven insights.

---

# 🎯 Business Problem

Organizations regularly conduct employee engagement surveys to understand workforce satisfaction and identify improvement opportunities.

However, raw survey data is difficult to interpret because it contains thousands of individual responses across multiple departments and employee roles.

This dashboard transforms the raw survey data into meaningful business insights by answering questions such as:

- Which departments have the highest engagement?
- Which employee roles are most satisfied?
- Which survey questions require leadership attention?
- Where are engagement gaps occurring?
- How complete and reliable is the survey data?

# 🧹 Data Cleaning (Power Query)

The dataset was transformed before visualization using Power Query.

- Removed duplicate survey responses
- Standardized question text
- Trimmed extra spaces
- Corrected inconsistent question labels
- Created Question Order column
- Built unified Role column
- Created Business Function grouping
- Replaced blank response values
- Added Favorability category
- Removed duplicate records

Some key findings from the analysis include:

- Certain departments consistently outperform the organizational average.
